## 문제 설명

문제
0보다 크거나 같은 정수 N이 주어진다. 이때, N!을 출력하는 프로그램을 작성하시오.

입력
첫째 줄에 정수 N(0 ≤ N ≤ 12)이 주어진다.

출력
첫째 줄에 N!을 출력한다.

## 해결 과정

재귀(자기 자신을 불러옴)함수를 이용한 해결방법도 있었다. 그 코드는 다음과 같다.
def factorial(n: int):
if n==0 or n==1:
return 1
esle:
return n\*factorial(n-1)

n = int(input())
print(factorial(n))

### 접근 방법

펙토리얼을 결과에 계속 곱하는 것. 반복문을 통해 해결했다.

## 코드

```python
N = int(input())
answer = 1
i = 1
while i <= N:
    answer *= i
    i += 1

print(answer)
```
