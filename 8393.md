# 문제 제목: 합

## 문제 정보

- **출처:** noj.am/8393
- **난이도:** Easy

## 문제 설명

n이 주어졌을 때, 1부터 n까지 합을 구하는 프로그램을 작성하시오.

## 해결 과정

for i in range문의 기능을 다시 한번 검증할 시간이라 생각했다.
다른 분의 풀이를 봤을때 점화식으로 과정을 압축하는 방법도 있었다.
더 나은 수학적 접근법이 있는지도 생각해볼 일이다.

### 접근 방법

반복문을 이용해 더하는 반복해 1씩 더하는 과정을 거친다.

## 코드

```python
n = int(input())
total = 0
for i in range(1, n+1):
    total += i
print(total)
```
